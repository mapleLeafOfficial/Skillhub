# 行为准则

## 我们的标准

贡献者和维护者应保持讨论的技术性、尊重性和建设性。

期望的行为：

- 聚焦问题、权衡和证据。
- 假定善意，但直接质疑薄弱的推理。
- 提供可操作的反馈。
- 尊重不同经验和领域知识水平。

不可接受的行为：

- 骚扰、侮辱或人身攻击
- 恶意争论或反复敌对
- 未经许可发布私人或敏感信息
- 阻碍 productive 协作的破坏性行为

## 执行

项目维护者有权删除评论、拒绝贡献或限制违反行为准则的参与者。

严重或屡次违规可能导致在项目空间中被临时或永久禁言。

## 举报

请通过私密渠道向维护者举报行为问题。不要使用公开 Issue 提交个人或敏感举报。
